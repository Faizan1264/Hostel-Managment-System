

# Hostel-Management
Manage Your Hostel 

## Topics Covered
- MERN Stack
- Hostel Management
- Bootstrap
- Reactstrap
- Redux
- JWT Authentication
- API testing using Jest

## Development and Installation

### Install Backend Dependencies
- `npm install`

### Install Frontend Dependencies
- `npm run client-install`

module.exports = {
  mongoURI: YOUR_LOCAL_MONGO_SERVER_URI,
	secretOrKey: YOUR_SECRET
}; 
```
### Run the application
- `npm run dev`

### For testing
- `npm run test`
